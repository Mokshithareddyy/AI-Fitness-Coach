before running install all requirements.txt
if you made any chnges after server connection make sure to STOP both frontend and backend servers.
Verify Flask-CORS Installation:
In backend/ terminal, with venv active: pip show Flask-CORS. Confirm it's there.
If not: pip install Flask-CORS and then pip freeze > requirements.txt.
Check backend/app.py CAREFULLY:
Ensure from flask_cors import CORS is at the top with other imports.
Ensure CORS(app, supports_credentials=True, resources={r"/api/*": {"origins": "http://localhost:8000"}}) is present immediately after app = Flask(__name__). No typos.
Save app.py.
Restart Backend Server:
In backend/ terminal (venv active)(venv\Scripts\activate): python app.py. Watch for any startup errors.
Restart Frontend Server:
In frontend/ terminal: python -m http.server 8000.
Test in Browser (Incognito/Private Window Recommended for this test):
Open http://localhost:8000.
Open Developer Tools (F12) -> Network Tab and Console Tab.
